## What is Velu?

![Velu overview diagram illustrating key integrations and workflows.](/img/1-290244d3.png)

Velu is an AI assistant that helps technical writers create and update documentation fast. It combines domain context, integrated project data, and style guidance so teams can focus on accuracy instead of manual drafting.

## Why Velu matters

- **Accelerates documentation cycles.** Velu generates first drafts and refreshes existing pages in minutes, keeping knowledge current without extra headcount.
- **Reduces context switching.** It pulls requirements, design notes, and user feedback from linked tools so writers stay in one workspace.
- **Protects team standards.** Velu applies the Microsoft Writing Style Guide, producing concise, plain-language content that matches your voice.

## Who Velu supports

Velu focuses on technical writers. It drafts and revises feature guides, release notes, and troubleshooting content so they can publish updates faster with less manual effort.

## Connected sources

Velu works alongside the systems your team already trusts. It can connect to:

- Project management tools for backlog items and release plans.
- Google Drive for specs, design docs, and stakeholder reviews.
- Knowledge bases like Confluence for institutional history.
- Support platforms to surface common questions and resolutions.
- Communication hubs such as Slack to capture product announcements and decisions.

## Core capabilities

1. **Draft generation.** Produce complete outlines or fully written topics based on live project inputs.
2. **Update automation.** Monitor linked sources and suggest precise doc edits when requirements evolve.
3. **Style alignment.** Enforce Microsoft Writing Style Guide patterns, terminology, and tone automatically.
4. **Collaboration loops.** Share AI-generated proposals with reviewers, track approvals, and apply changes in one place.

## Built for responsible adoption

Velu highlights the origin of every suggestion, so reviewers can verify facts quickly. It keeps customer data in your connected systems and uses integration-level permissions to respect workspace privacy.
